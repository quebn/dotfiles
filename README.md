# My Dotfiles
My Linux dotfiles themed around Rose Pine with personal tweaks

> [!CAUTION]
> This repository depends on some privates files and symlinks.\
> Clone at your own risk.

## What I use ATM.
- Tiling Window Manager: [Hyprland](https://hyprland.org/)
- Status Bar and Notifications: [Quickshell](https://git.outfoxxed.me/quickshell/quickshell)
- Text Editor: [Neovim](https://github.com/neovim/neovim)
- Terminal: [Kitty](https://github.com/kovidgoyal/kitty)
- Shell: [Fish](https://fishshell.com/)
- App Launcher: [Wofi](https://github.com/davatorium/rofi)
